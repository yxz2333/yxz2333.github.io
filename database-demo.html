JSON 文档存储

本项目用 JSON 替代传统关系型数据库

index.json — 索引表 {slug}.json — 数据表 aliases.json — 映射表 code_embeddings — 向量库 recommendations — 推荐 macros.json — 配置表

7 张"表" · 纯静态 · 零数据库依赖

七张表总览

文件等价关系表由谁生成存放位置
index.json索引表parse_notes.pyfrontend/public/{p,k,c,other}/
{slug}.json数据表（每篇一行）parse_notes.pyfrontend/public/{p,k,c,other}/
aliases.json别名映射表parse_notes.pyfrontend/public/{p,k,c,other}/
code_embeddings.json向量元信息表recommend.pyfrontend/public/ + notes-out/
code_embeddings.npy向量矩阵recommend.pyfrontend/public/ + notes-out/
recommendations.json推荐结果表recommend.pyfrontend/public/ + notes-out/
macros.json配置表手动backend/

所有文件双写：notes-out/ 后端留底 + frontend/public/ 前端引用

index.json — 笔记索引表

顶层字段

字段类型说明
generated_atstring生成时间 ISO 8601
countint笔记总数
type_directorystring所属目录 p/k/c/other
notesarray笔记条目数组

notes[i] 字段

字段类型说明
slugstring唯一 URL 标识
titlestring笔记标题
typestring题目/知识点/比赛/其它
tagsstring[]标签列表
difficultynumber|null难度 1-5
summarystring摘要
parentstring|null父页面 slug
created_atstring创建日期
updated_atstring修改日期

每个类型目录各一份，前端按需加载 ~30KB

{slug}.json — 笔记正文

核心字段

字段类型说明
slugstring唯一标识
titlestring标题
typestring类型
formatstringlogseq/markdown/html
sectionsarray结构化 section 树
code_languagesstring[]代码语言列表
sourcestring来源（HTML）
aliasesstring[]别名列表

sections 类型

heading标题 + level + 子 blocks
p段落，html 字段 + depth
code代码块，lang + code 字段
math公式，latex 字段
table表格，rows 数组
html纯 HTML 页面

aliases.json — 别名映射表

{"重链剖分": "树链剖分", "长链剖分": "树链剖分", "最小生成树": "生成树", "01背包": "背包", "记搜": "记忆化搜索"}

key = 别名 slug  |  value = 规范 slug

前端 Tags / TagArticles / Note 三处查到别名后跳转到对应知识点

code_embeddings — 向量存储

code_embeddings.json（元信息）

字段类型说明
note_slugstring笔记 slug
note_titlestring笔记标题
note_routestring前端路由 /k/xxx
block_indexint第几个代码块
code_hashstringSHA256 前 12 位
langstringcpp / c++

code_embeddings.npy（向量矩阵）

属性值
dtypefloat32
shape(N, 768)
第 i 行blocks[i] 的 UnixCoder 向量

JSON 不存 embedding——768 维浮点太膨胀
分离存储：JSON 元信息 + npy 紧凑矩阵
索引严格对齐，增删后重建

recommendations.json — 推荐结果

{"generated_at": "2026-06-16T...", "model": "microsoft/unixcoder-base", "top_k": 6, "recommendations": { "cf2233c::0": [ {"slug":"括号匹配", "similarity":0.91}, {"slug":"栈与括号", "similarity":0.78} ], "cf2233c::1": [ ... ] }}

key = {slug}::{block_index}  |  value = Top-K 推荐数组

前端 Note.vue 按当前 slug 前缀过滤加载
